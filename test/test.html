JFN Examples


For docs see here

Example: simple

data:
{
  "foo": "bar",
  "bar": 3
}
 schema:
{
  "type": "object",
  "properties": {
    "foo": {
      "type": "string",
      "enum": [
        "foo",
        "bar"
      ],
      "title": "Your title"
    },
    "bar": {
      "type": "integer",
      "description": "integers are nice"
    }
  }
}
Example: type hinting
data:
{
  "foo": "bar",
  "bar": 3
}
 schema:
{
  "type": "object",
  "properties": {
    "foo": {
      "type": "string",
      "typehint": "rich"
    }
  }
}
